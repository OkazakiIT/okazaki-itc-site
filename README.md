# 個人事業主 名刺サイト

個人事業主の「名刺代わり」となる1ページ完結・モバイルファーストの静的サイト。
バックエンド無し（HTML / CSS / JS のみ）で、AWS S3 + CloudFront で配信予定。

## ディレクトリ構成

```
business_card_site/
├── index.html          … サイト本体（1ページ完結）
├── css/
│   └── style.css       … 共通スタイル（デザイントークンで一元管理）
├── js/
│   └── main.js         … 共通スクリプト（ナビ開閉・スクロール演出）
├── images/             … 公開用の画像（現在はSVGプレースホルダー）
└── client_source/      … ★依頼主からの素材置き場（公開対象外・デプロイしない）
    ├── pdf/            …   テキスト元のPDF
    └── images/         …   顔写真・ロゴ・実績写真など
```

## デプロイ対象 / 対象外

- **アップロードする**: `index.html` / `css/` / `js/` / `images/`
- **アップロードしない**: `client_source/`（素材の元データ。S3には含めない）
